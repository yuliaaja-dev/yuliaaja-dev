# Pong PWA

Progressive Web App (PWA) sederhana bertema Pong dengan mode **Player vs AI**, dibuat menggunakan HTML, CSS, dan JavaScript tanpa framework.

---

## Fitur

- Player vs AI
- Responsive (Desktop & Mobile)
- Progressive Web App (PWA)
- Dapat diinstal
- Offline menggunakan Service Worker
- Floating Popup CSS (Win/Lose)
- Tanpa library eksternal
- Kode mudah dikembangkan

---

## Struktur Folder

```
/
│
├── index.html
├── style.css
├── script.js
├── manifest.json
├── sw.js
├── offline.html
├── LICENSE
├── README.md
│
└── images/
    └── icons/
        ├── icon-192.png
        └── icon-512.png
```

---

## Cara Menjalankan

1. Letakkan semua file sesuai struktur folder.
2. Jalankan menggunakan server lokal atau hosting.
3. Buka `index.html`.
4. Browser akan mendaftarkan Service Worker secara otomatis.
5. Install aplikasi melalui browser jika tersedia.
